# 🌱 Spring Boot Projects Collection

### (Java | Spring Boot | IntelliJ IDEA)

## 📌 Description

This repository contains multiple **Spring Boot applications and experiments** developed using **IntelliJ IDEA**.

It represents a progression from basic Spring Boot setup to **full-stack backend systems**, including advanced implementations with:

* Entity-Response separation
* Lombok integration
* Structured API design

## ⚙️ How It Works

Each project follows **Spring Boot layered architecture**:

* **Controller Layer** → Handles API requests
* **Service Layer** → Business logic
* **Repository Layer** → Database operations
* **Model Layer** → Entities / DTOs

## 🧑‍💻 How to Run

### 🔧 Prerequisites

* Java JDK 8+
* Maven
* IntelliJ IDEA

---

### ▶️ Run Using IntelliJ

1. Open project in IntelliJ IDEA
2. Select any module
3. Run `main` class (Spring Boot Application)

---

### ▶️ Run Using Terminal

```bash
mvn spring-boot:run
```
